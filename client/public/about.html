About

Toronto Weekly Events is a map of recurring weekly events happening across the city. Unlike one-off event calendars, this focuses on reliable, community-driven gatherings you can count on week after week.

This project was inspired by ChrisD's Toronto Events Calendar and a desire to make recurring events easier to discover. The idea: open it on any day, at any time, and see what's happening nearby.

Event submissions are curated. If you know of a weekly event that should be included, contact me (Erin Saint Gull) on Twitter or any other platform. Especially if you're someone in the Friendly Ambitious Nerds, TPOT, tech, or meditation communities.
